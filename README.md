# Hive GUI

A Roblox GUI library for executors and Studio.

## Usage

```lua
local Hive = loadstring(game:HttpGet("https://raw.githubusercontent.com/utility-inc/unity-inc/main/library.lua"))()

local GUI = Hive.new("MyScript")

GUI:SetToggleKey(Enum.KeyCode.RightShift)

GUI:Toggle()
```

## Compatibility

- Works in Roblox Studio
- Works in executors
